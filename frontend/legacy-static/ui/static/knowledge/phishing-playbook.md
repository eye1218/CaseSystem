# 钓鱼邮件处置作业卡

## 目标

帮助值守人员在较短时间内完成钓鱼邮件事件的确认、影响面评估和客户反馈同步。

## 基础检查

- 确认邮件主题、发件人、投递时间
- 确认是否存在点击、下载、执行行为
- 确认受影响邮箱数量与终端数量

## 处置顺序

### 1. 邮件侧

- 尝试撤回或隔离同源邮件
- 标记 IOC 并同步邮件网关规则

### 2. 终端侧

- 检查点击终端是否访问恶意域名
- 检查是否触发下载、执行或凭据输入

### 3. 客户侧同步

> 对客户公开评论中，只同步已确认事实，不暴露内部研判备注。

- 说明当前影响范围
- 说明是否已完成隔离
- 说明下一步预计动作

## 关闭前确认

- 恶意邮件已处置
- 受影响终端已确认状态
- 客户已收到结论或报告链接
