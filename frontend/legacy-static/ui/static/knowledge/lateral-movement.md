# 横向移动初判与隔离指引

## 适用范围

- 工单分类为**安全事件**
- 已出现多主机登录扩散迹象
- 需要在短时间内给出隔离和升级判断

## 优先动作

- 先确认告警源主机是否仍在线
- 优先隔离源主机与已确认受影响资产
- 记录首个异常登录时间与最近一次横向访问时间

## 研判步骤

### 1. 账户与主机双维确认

- 确认异常账户最近 24 小时登录轨迹
- 确认目标主机是否与源主机存在共享凭据使用
- 对比管理网、办公网、服务器区的访问差异

### 2. 风险提升条件

- 出现域控相关访问
- 出现批量 SMB / RDP 登录失败后成功
- 出现横向工具投放痕迹

### 3. 升级建议

> 若出现跨网段扩散迹象，优先升级给指定专家确认。

- 未指定具体专家时，可升级到目标池子
- 已指定专家时，进入升级待确认子状态

## 留痕要求

```text
记录隔离时间
记录首个确认受影响资产
记录升级申请时间
```
